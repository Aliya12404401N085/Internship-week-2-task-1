# Internship-week-2-task-1
# Week 2 Task 1 - Data Cleaning and Exploratory Data Analysis

## Objective
Perform data cleaning and exploratory data analysis on a dataset using Python.

## Dataset
Titanic Dataset

## Tasks Completed

- Data loading using Pandas
- Missing value handling
- Duplicate removal
- Data cleaning
- Exploratory Data Analysis
- Correlation analysis
- Visualization generation

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
## Results

### Data Cleaning
- Missing Age values filled with median
- Missing Embarked values filled with mode
- Missing Cabin values replaced with "Unknown"
- Duplicate rows removed

### Visualizations
1. Age Distribution Histogram
2. Age vs Fare Scatter Plot
3. Correlation Heatmap

### Insights
- Most passengers were between 20–40 years old.
- Fare generally increases with passenger class.
- Survival has correlation with passenger class and fare.
